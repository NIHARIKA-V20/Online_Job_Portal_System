Online Job Portal System

The Online Job Portal System is a web application designed to connect job seekers and employers. Users can register, log in, search for jobs, apply for jobs, and manage their profiles. Employers can post job vacancies, view applications, and manage job listings. The system is built using Node.js, Express.js, and React.js.

Features

For Job Seekers:
Create an account and log in
Search and filter job opportunities
View job details
Apply for jobs
Manage profile information

For Employers:
Register and log in
Post job openings
Edit or delete job postings
View applications received

General Features:
Responsive user interface
Secure authentication
Clean and organized folder structure
API-driven architecture

Installation and Setup

Step 1: Clone the repository
git clone https://github.com/NIHARIKA-V20/Online_Job_Portal_System.git
cd Online_Job_Portal_System

Step 2: Install all dependencies
npm install

Step 3: Run the development server
If frontend: npm start
If backend: nodemon server.js or node app.js

How the System Works:
Users sign up as job seekers or employers
Employers create and publish job openings
Job seekers search for jobs and submit applications
The system manages user authentication and job-related data
